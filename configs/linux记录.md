



```
yum info nginx 查看软件版本信息
rpm -ql nginx 查看目录
/etc/yum.repos.d/   源目录
nginx -t 检查nginx配置是否有语法错误
which nginx 查看软件安装目录
nginx  启动nginx
nginx -s stop  停止nginx


ss -nltp  查看已经启动的服务

ps aux 查看进程
ps axo pid,cmd,psr| grep nginx 查看进程所在的CPU


yum history  查看最近安装的包
yum history info 9 查看最近安装的9个包的详细信息


cd /run  正在运行的程序生成的pid所在目录
```
